# tGHSX Stablecoin Protocol

A crypto-collateralized GHS-pegged stablecoin deployed on Polygon, using Chainlink, Supabase, and smart contracts.

> — Proverbs 16:3 (Common Ground Bible)
## Architecture Summary

- ERC-20 Token (`tGHSX`) on Polygon
- Overcollateralized via ETH, BTC, USDT
- Oracle pricing via Chainlink
- Supabase for auth and off-chain data
- React frontend with Supabase Auth
- FastAPI/Express backend
